# CV Interactivo – Jessica Solar

Currículum Vitae interactivo desarrollado con HTML, CSS, Bootstrap, JavaScript y jQuery.

## Descripción
Este proyecto es una aplicación web desarrollada con HTML, CSS y JavaScript, enfocada en reforzar los conceptos fundamentales del desarrollo frontend.

Incluye manipulación del DOM, uso de eventos, estilos responsivos y funcionalidades interactivas que mejoran la experiencia del usuario. El objetivo principal del proyecto es aplicar buenas prácticas de estructura, diseño y comportamiento dinámico en una página web funcional.

## Tecnologías
- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery
